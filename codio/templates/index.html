What is Codio?

Codio is an online coding platform which allows developers to share their awesome code with other developers. We help people write better, more optimised code faster.

Who made this?

This online service has been created by college students. This was a hobby project, we want to help developers find good code faster.

How do I get started?

You can follow the following steps to start quickly :
  1. Create a new Account.
  2. Go to the Submit Code page.
  3. Fill all the required fields for Title, Language, code etc.
  4. Submit the code and share the link with your friends.
You can also go to the view code page to view code submitted by other developers, you can search,filter and sort the code easily.



So Ready to get started Submit your code or View code submitted by others!
